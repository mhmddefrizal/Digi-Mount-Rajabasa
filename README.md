# 🚀 Digi-Mount Rajabasa

Sistem registrasi pendakian Gunung Rajabasa berbasis rombongan yang memungkinkan pendataan anggota, validasi keselamatan, pencatatan logistik, perhitungan biaya, serta penyediaan tiket digital sederhana sebagai bukti pendakian.

---

## 🎯 Tujuan

Aplikasi ini dibuat untuk:

- Mempermudah proses registrasi pendakian
- Meningkatkan keselamatan pendaki
- Mengelola data pendaki secara digital
- Mengontrol logistik dan sampah pendaki

---

## 🧩 Fitur Utama

- 🔐 Authentication (Login & Register)
- 📝 Pendaftaran pendakian (berbasis rombongan)
- 👥 Pendataan anggota (NIK, nama, no HP)
- ⚠️ Validasi keselamatan (headlamp & jas hujan wajib)
- ⏰ Validasi waktu (tektok tidak boleh di atas jam 12)
- 🎒 Input logistik (opsional upload foto)
- 🗑️ Input sampah setelah pendakian
- 💰 Perhitungan biaya otomatis
- 🎟️ Tiket digital sederhana
- 📋 Riwayat pendaftaran
